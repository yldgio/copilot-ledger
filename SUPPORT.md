# Support

Use GitHub issues for support requests, bug reports, documentation gaps, and feature proposals.

## Before asking for help

Check:

- `README.md` for install and usage instructions
- `AUTO_INSTALL.md` for agent install instructions
- `copilot-ledger-design.md` for implementation details
- Existing issues for known problems

## What to include

- Copilot CLI version
- Operating system
- Whether the extension is installed for the user or workspace
- The command or workflow that failed
- Expected behavior and actual behavior
- Redacted logs or error messages

Do not share secrets, private prompts, or raw ledger records in public issues.

## Security issues

Report vulnerabilities using `SECURITY.md`, not public issues.
